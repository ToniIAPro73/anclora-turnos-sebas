# Product Specification: Anclora Turnos Sebas v0

## Vision
"Anclora Turnos Sebas" is a personal productivity application designed to manage work shifts effectively. It provides a visual, intuitive interface to track time, calculate weekly hours, and predict upcoming shifts.

## General Objectives
- **Simplicity**: No complex enterprise features. Focus on personal use.
- **Speed**: Fast interaction for adding and checking shifts.
- **Visual Clarity**: Premium design following Antigravity principles.

## Target Audience
- Individual users managing variable work schedules (Sebas).

## Non-Functional Requirements
- **Local First**: Data should be available offline (localStorage or similar).
- **Responsive**: Mobile and desktop support.
- **Type Safety**: Built with TypeScript.
- **Clean Architecture**: Separation of UI and business logic.
